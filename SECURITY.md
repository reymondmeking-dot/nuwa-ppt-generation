# Security Policy

## Secrets

Do not commit API keys, provider tokens, cookies, SSH keys, or generated `.env` files.

The repository may mention environment variable names such as `GEMINI_API_KEY` or `OPENAI_API_KEY` in documentation, but real values must stay outside git.

## Reporting

If you find a security issue, open a private report with enough detail to reproduce it. Do not publish exploitable credentials or user data in public issues.
